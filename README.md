# GIFDesk

Lightweight application that allows you to place animated images directly on your desktop<br>
Supports all types of `GIF` `APNG` `WEBP`

## Basic information

Utility displays transparent animation windows. You can freely move any window by holding the left mouse button.

Each window can stay on top of all others like a desktop sticker. You can also adjust playback speed and window scale, make it hide automatically when hovered, or let mouse clicks pass through to the window behind.
